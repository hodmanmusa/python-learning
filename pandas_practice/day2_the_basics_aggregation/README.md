## Project Overview

This project is part of my **Pandas learning journey**.
The goal is to practice and consolidate core Pandas concepts by performing an exploratory data analysis (EDA) on the popular Netflix Movies and TV Shows dataset.

The project focuses on:

* Understanding the structure of the dataset
* Applying aggregation, grouping, and pivot tables
* Exploring trends in Netflix content over time
* Practicing basic data visualization with Matplotlib

This is a **learning-focused project**, not a production-grade analysis.

--- 

## Dataset

* **Name:** `netflix_titles.csv`
* **Content:** Netflix Movies and TV Shows
* **Key columns used:**

  * `type` (Movie / TV Show)
  * `release_year`
  * `country`
  * `title`

---

## Concepts Practiced

### Pandas

* Loading and inspecting data
* Filtering and sorting data
* Creating new columns (`content_age`, `is_recent`)
* Summary statistics (mean, median, min, max)
* Aggregation with `agg()`
* Counting with `value_counts()` (including normalized counts)
* Grouped analysis using `groupby()`
* Pivot tables (`index`, `values`, `aggfunc`, `fill_value`)
* Cumulative statistics (`cumsum`)
* Explicit indexing (`set_index`, `reset_index`)
* Row and column selection using `.loc`

### Matplotlib

* Line charts
* Bar charts
* Histograms
* Scatter plots
* Adding titles and axis labels
